文本组件是 Android 开发中用于显示文本的组件。

## 你将学到：
- 项目设置与场景管理
- 物理效果与玩家控制
- 摄像机脚本与跟随系统
- 碰撞检测与拾取物体
- UI 实现与分数显示

## 前置条件：
- 已安装 Android Studio 4.1.1 或更高版本
- 具备 Java 的基础理解（有帮助但非必需）
- 30–60 分钟的空闲时间
