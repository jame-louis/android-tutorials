## Setting Up Your Development Environment

Before diving into Unity, let's ensure your development environment is properly configured for the best learning experience.

### System Requirements
- **OS:** Windows 10+, macOS 10.15+, or Ubuntu 20.04+
- **RAM:** Minimum 8GB (16GB recommended)
- **GPU:** DirectX 11 or OpenGL 4.5 compatible
- **Storage:** 5GB free space for Unity + project files

### Installing Unity Hub
1. Download Unity Hub from [unity.com/download](https://unity.com/download)
2. Install Unity Hub following the on-screen instructions
3. Open Unity Hub and sign in with your Unity ID
4. Install Unity Editor version 2022.3 LTS

<div class="info-box warning">
  <strong>Important:</strong> Make sure you have the latest version of Unity Hub installed before proceeding.
</div>
