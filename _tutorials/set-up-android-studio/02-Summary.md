## 摘要

- 集成开发环境（IDE）是一组用于开发软件的工具。
- Android Studio 是基于 IntelliJ IDEA 的IDE，用于开发 Android 应用。

## 教程项目工程配置的相关版本

- Android Studio 4.1.1, 2020.11.05。
- Android SDK: 30.0.0, Android 11.0 (R), API 30, 
- Android Gradle Plugin version: 4.1.1
- Gradle version: 6.5
